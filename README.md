# Portfolio Website

A modern and responsive personal portfolio website built using **React** and **Tailwind CSS** to showcase my skills, projects, and professional journey. The portfolio also integrates **EmailJS** to enable a functional contact form without requiring any backend, making it lightweight and efficient.

## Key Highlights
- Built with React (component-based architecture)  
- Styled using Tailwind CSS  
- Responsive design for all screen sizes  
- Contact form powered by EmailJS  
- Projects, skills, and about sections  
- Clean and minimal UI
